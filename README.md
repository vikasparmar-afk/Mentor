# Sanskriti: Your Journey Through Indian Philosophy

Embark on a transformative exploration of ancient wisdom with Sanskriti, your dedicated companion for studying Indian philosophy. This comprehensive reading tracker and knowledge repository helps you navigate the profound depths of Vedanta, Yoga, Samkhya, Buddhism, and other classical Indian philosophical traditions.

## Features

### Discover the Eternal Truths
- Track your progress through timeless texts like the Bhagavad Gita, Upanishads, and Yoga Sutras
- Monitor your reading habits and maintain consistent philosophical study
- Organize your learning journey across different schools of Indian thought
- Reflect on profound teachings with personalized notes and insights

### Features for Deep Learning
- Progress tracking for each philosophical text
- Category-based organization (Vedanta, Yoga, Samkhya, Buddhism, Jainism, and more)
- Personal rating system for books and concepts
- Reading streaks and consistency metrics
- Insightful analytics on your learning patterns
- Note-taking for profound realizations

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Build

```bash
npm run build
```

### Production

```bash
npm start
```

## Technology Stack

- **Next.js 15** - React framework with App Router
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Modern utility-first styling
- **Local Storage** - Client-side data persistence

## Project Structure

```
sanskriti/
├── app/                    # Next.js app directory
│   ├── page.tsx           # Dashboard
│   ├── library/           # Library page
│   ├── add/               # Add book page
│   ├── analytics/         # Analytics page
│   └── book/[id]/         # Book details page
├── components/            # Reusable React components
├── lib/                   # Utility functions and storage
├── types/                 # TypeScript type definitions
└── public/               # Static assets
```

## Usage

1. **Dashboard**: View your reading overview, streaks, and quick stats
2. **Library**: Browse all your philosophical texts with filtering options
3. **Add Book**: Add new books to your collection
4. **Book Details**: Track progress, add notes, and update reading status
5. **Analytics**: View insights and statistics about your reading journey
